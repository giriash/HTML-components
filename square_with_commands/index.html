Using following commmands to control the square:

GO # : move the block # steps following current direction. example: GO 3

TUN LEF # : change the direction to left #times, example: TUN LEF

TUN RIG # : change the direction to right #times, example: TUN RIG

TUN BAC # : change the diraction to back #times, example: TUN BAC

TRA LEF # : move the block to left without changing direction, example: TRA LEF 2

TRA RIG # : move the block to right without changing direction, example: TRA RIG 2

TRA BOT # : move the block to bottom without changing direction, example: TRA BOT 2

TRA TOP # : move the block to top AND change the direction, example: TRA TOP 2

MOV LEF # : move the block to left AND change the direction, example: MOV LEF 2

MOV RIG # : move the block to right AND change the direction, example: MOV RIG 2

MOV BOT # : move the block to bottom AND change the direction, example: MOV BOT 2

MOV TOP # : move the block to top AND change the direction, example: MOV TOP 2

BUILD : build a wall in front of the block, example: BUILD

BRU color : color the front wall with the color, example: BRU RED
